Theme Information

The theme toggle system includes:

  • CSS Variables: All colors are defined as CSS custom properties for easy theming
  • JavaScript Class: ThemeToggle class handles all theme switching logic
  • Local Storage: User preferences are remembered across sessions
  • System Detection: Automatically detects user's system theme preference
  • Accessibility: Proper ARIA labels and keyboard navigation support
